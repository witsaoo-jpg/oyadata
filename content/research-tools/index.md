# Research Tools (เครื่องมือวิจัย)

รวบรวมเครื่องมือทางสถิติที่จำเป็นสำหรับการทำวิจัยทางการพยาบาลและการวิเคราะห์ข้อมูลเบื้องต้น
# เครื่องมือที่ใช้ทดสอบสมมติฐาน

## 1. การทดสอบแบบพาราเมตริก (Parametric Tests)

### 1.1 การเปรียบเทียบค่าเฉลี่ย
- **t-test** (การทดสอบที)
  - One-sample t-test
  - Independent samples t-test
  - Paired samples t-test
- **Z-test** (การทดสอบ Z)
- **ANOVA** (การวิเคราะห์ความแปรปรวน)
  - One-way ANOVA
  - Two-way ANOVA
  - Repeated measures ANOVA
- **ANCOVA** (การวิเคราะห์ความแปรปรวนร่วม)
- **MANOVA** (การวิเคราะห์ความแปรปรวนแบบหลายทาง)

### 1.2 การวิเคราะห์ความสัมพันธ์และการพยากรณ์
- **Linear Regression** (การถดถอยเชิงเส้น)
- **Multiple Regression** (การถดถอยเชิงพหุ)
- **Logistic Regression** (การถดถอยโลจิสติก)
- **Path Analysis** (การวิเคราะห์เส้นทาง)
- **Structural Equation Modeling (SEM)** (โมเดลสมการโครงสร้าง)

### 1.3 การทดสอบสมมติฐานเฉพาะในโมเดล
- **F-test** (การทดสอบ F)
- **Wald Test**
- **Likelihood Ratio Test (LRT)**
- **Score Test (Lagrange Multiplier Test)**

## 2. การทดสอบแบบไม่พาราเมตริก (Non-parametric Tests)

### 2.1 สำหรับข้อมูลอันดับหรือไม่ทราบการแจกแจง
- **Mann-Whitney U Test**
- **Wilcoxon Signed-Rank Test**
- **Kruskal-Wallis Test**
- **Friedman Test**
- **Sign Test**

### 2.2 สำหรับข้อมูลเชิงพรรณนา (Categorical Data)
- **Chi-square Test** (การทดสอบไคสแควร์)
  - Chi-square goodness-of-fit
  - Chi-square test of independence
- **Fisher’s Exact Test**
- **McNemar’s Test** (สำหรับข้อมูล paired nominal)
- **Cochran’s Q Test**

## 3. การตรวจสอบสมมติฐานพื้นฐาน (Assumption Checks)

- **การทดสอบการแจกแจงปกติ (Normality Tests)**
  - Shapiro-Wilk Test
  - Kolmogorov-Smirnov Test
- **การทดสอบความเท่ากันของความแปรปรวน (Homogeneity of Variance)**
  - Levene’s Test
  - Bartlett’s Test
- **การทดสอบความแปรปรวนคงที่ (Homoscedasticity)**
  - Breusch-Pagan Test
  - White Test
- **การทดสอบอิสระของส่วนเหลือ (Independence of Errors)**
  - Durbin-Watson Test
- **การตรวจสอบ Multicollinearity**
  - Variance Inflation Factor (VIF)

## 4. วิธีการทางเลือกและขั้นสูง

- **Bootstrap Methods**
- **Permutation Tests**
- **Bayesian Hypothesis Testing**
- **Confirmatory Factor Analysis (CFA)**

## 5. มาตรวัดและสถิติสนับสนุน

- **p-value**
- **Confidence Interval (CI)**
- **Effect Size** (เช่น Cohen’s d, Eta-squared, Odds Ratio)
- **Statistical Power**
- **R-squared / Adjusted R-squared**
- **Goodness-of-fit Indices** (เช่น CFI, TLI, RMSEA – ใช้ใน SEM/CFA)
